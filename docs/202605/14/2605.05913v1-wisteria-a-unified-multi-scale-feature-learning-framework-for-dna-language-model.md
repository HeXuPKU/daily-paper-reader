## 摘要
DNA 语言模型旨在通过捕获 DNA 序列中的长程依赖关系，来破译基因组的调控语法和语义。现有方法强调长程 token 交互，但往往忽略了局部基序（motifs）与全局依赖之间的相互作用。在本文中，我们提出了 Wisteria，这是一种在统一框架内整合了 DNA 序列多尺度特征学习的基因组语言模型。具体而言，Wisteria 通过门控空洞卷积增强了基于 Mamba 的架构，以捕获局部基序和调控模式，同时利用门控多层感知器细化全局依赖。我们还引入了一种基于傅里叶的注意力机制，以支持频域建模、周期性扩展和长度泛化。在包含短程和长程依赖的四种实验设置中，Wisteria 在下游基准测试中表现出优于竞争性 DNA 语言模型基准的强劲性能。这些结果表明，Wisteria 有效地统一了局部和全局依赖建模，适用于多尺度基因组序列分析。

## Abstract
DNA language model aims to decipher the regulatory grammar and semantic of genomes by capturing long range dependencies in DNA sequences. Existing methods emphasize long range token interactions but often ignore the interplay between local motifs and global dependencies. In this paper, we propose Wisteria, a genomic language model that integrates multi scale feature learning within a unified framework for DNA sequence. Specifically, Wisteria augments the Mamba based architecture with gated dilated convolutions to capture local motifs and regulatory patterns, while gated multilayer perceptrons refine global dependencies. We further introduce a Fourier based attention mechanism to support frequency domain modeling, periodic extension and length generalization. Across four experimental settings with both short and long range dependencies, Wisteria demonstrates strong performance on downstream benchmarks against competitive DNA language model baselines. These results indicate that Wisteria effectively unifies local and global dependency modeling for multi scale genomic sequence analysis.

#### 1. 核心问题与整体含义（研究动机和背景）
DNA 序列建模面临一个核心挑战：**多尺度依赖的平衡**。基因组调控既依赖于局部的连续基序（Motifs，如启动子、剪接位点，通常跨度数十到数百个碱基对），也依赖于长程的相互作用（如增强子、变异效应预测，跨度可达数千甚至上百万个碱基对）。
*   **现有局限**：传统的 CNN 擅长局部特征但难以处理长程依赖；Transformer 具有全局视野但计算复杂度高且缺乏局部归纳偏置；新兴的 Mamba（状态空间模型）虽然高效，但在精细的局部特征提取上表现稍逊。
*   **周期性信号**：DNA 具有约 10-11 bp 的内在周期性（与核小体定位相关），现有的旋转位置编码（RoPE）在处理超长序列扩展时会产生频谱损伤。
*   **研究目标**：提出一个统一框架，同时高效捕获 DNA 的局部基序、全局依赖以及频域周期性特征。

#### 2. 方法论
Wisteria 采用了一种自下而上的分层架构，主要包含以下核心组件：
*   **GCMB 模块（Gated Convolution–BiMamba）**：
    *   **双向 Mamba**：负责捕获上游和下游的全局上下文。
    *   **门控空洞卷积（Gated Dilated Convolutions）**：在浅层引入不同扩张率的深度卷积，通过门控机制（Sigmoid 激活）提取多尺度的局部基序特征。
*   **门控 MLP 模块（Gated MLP）**：在 GCMB 之后进行非线性特征融合与精炼，利用门控线性单元（SiLU 激活）增强特征的选择性。
*   **傅里叶注意力机制（Fourier-based Attention）**：
    *   在顶层引入 **FoPE（Fourier Position Embedding）**。
    *   **核心思想**：将位置信息表示为具有多个谐波频率的傅里叶级数，而非 RoPE 的单频率旋转。这增强了模型对周期性信号的建模能力，并解决了长序列扩展时的频谱泄漏问题。
