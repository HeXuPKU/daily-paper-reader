## 摘要
多模态大语言模型在临床视觉问答任务中具有广阔前景，但由于高昂的计算成本，将其扩展到 3D 成像领域受到了阻碍。先前的方法通常依赖于 2D 切片或固定长度的 token 压缩，这破坏了体积连续性并掩盖了细微的发现。我们提出了 Photon，这是一个使用变长 token 序列表示 3D 医学体数据的框架。Photon 引入了指令条件下的 token 调度和代理梯度传播，以便在训练和推理过程中自适应地减少 token，从而在降低计算成本的同时减轻由冗余 token 引起的注意力稀释。它结合了带有梯度恢复的自定义反向传播规则，以便在存在离散 token 丢弃的情况下实现可微优化。为了稳定 token 压缩并确保视觉证据的可靠使用，Photon 进一步应用了正则化目标，以减轻纯语言偏见并提高可靠性。在多种医学视觉问答任务上的实验表明，Photon 在降低资源消耗并加速训练和推理的同时，实现了最先进的准确率。

## Abstract
Multimodal large language models are promising for clinical visual question answering tasks, but scaling to 3D imaging is hindered by high computational costs. Prior methods often rely on 2D slices or fixed-length token compression, disrupting volumetric continuity and obscuring subtle findings. We present Photon, a framework that represents 3D medical volumes with token sequences of variable length. Photon introduces instruction-conditioned token scheduling and surrogate gradient propagation to adaptively reduce tokens during both training and inference, which lowers computational cost while mitigating the attention dilution caused by redundant tokens. It incorporates a custom backpropagation rule with gradient restoration to enable differentiable optimization despite discrete token drop. To stabilize token compression and ensure reliable use of visual evidence, Photon further applies regularization objectives that mitigate language-only bias and improve reliability. Experiments on diverse medical visual question answering tasks show that Photon achieves state-of-the-art accuracy while reducing resource usage and accelerating both training and inference.

### 2. 方法论：核心思想与关键技术
Photon 框架的核心在于实现**变长 Token 序列**的自适应处理，主要包含以下技术：
*   **指令调节的 Token 调度（ITS）**：
    *   **指令感知显著性估计（ISE）**：利用指令 Token 之间的自亲和力（Self-affinity）确定核心指令语义，再通过指令与视觉 Token 的交叉注意力计算每个视觉 Token 的显著性得分。
    *   **实例感知阈值预测器（ITP）**：根据当前影像的复杂度和指令需求，动态预测一个概率阈值 $\theta$，而非使用固定比例，从而决定保留哪些 Token。
*   **代理梯度传播（SGP）**：
    *   **解决不可微问题**：离散的 Token 丢弃操作在反向传播中不可导。Photon 引入了“直通估计器”（Straight-through estimator），在推理时进行硬剪枝，在训练时保留概率掩码。
    *   **梯度恢复**：利用激活值与梯度的乘积作为代理信号，为被丢弃的 Token 构造代理梯度，确保阈值预测器能够根据任务损失进行端到端优化。
*   **正则化目标**：
    *   **软保留带宽限制**：防止模型走向极端（保留全部或剪掉全部 Token）。
    *   **鲁棒性与掩码翻转正则化**：通过引入噪声和对抗性掩码，强制模型依赖视觉证据而非文本先验，减少“幻觉”和语言偏见。

### 6. 主要结论与发现
*   **性能领先**：Photon 在 3D-RAD 和 DeepTumorVQA 上均取得了 SOTA 结果，尤其在需要精确空间理解的“医学测量”和“病灶计数”任务中提升显著（部分指标提升超 35%）。

### 7. 优点与亮点
*   **变长表示**：打破了固定 Token 长度的限制，实现了真正的“按需分配”计算资源。
*   **训练推理双加速**：不同于许多仅在推理阶段剪枝的方法，Photon 的 SGP 机制使其在训练阶段也能享受剪枝带来的提速。
*   **指令敏感性**：模型能根据问题（如问肺部还是问肾部）动态调整保留的解剖区域，具有极强的临床解释性。

### 8. 不足与局限
*   **微小病灶挑战**：在极小病灶（如 4mm 结节）的定量测量上仍存在一定误差，反映了 MLLM 在极细粒度数值推理上的固有难题。
*   **长尾分布影响**：对于训练集中罕见的解剖结构或罕见病例，Token 调度器的显著性估计可能不够准确，存在误删风险。
*   **临床验证**：目前主要基于公开数据集评估，尚未在真实的实时临床工作流中进行前瞻性验证。
