## 摘要
多模态大语言模型在临床视觉问答任务中具有广阔前景，但由于高昂的计算成本，将其扩展到 3D 成像面临挑战。先前的方法通常依赖于 2D 切片或固定长度的 token 压缩，这破坏了体积连续性并掩盖了细微的发现。我们提出了 Photon，这是一个使用变长 token 序列表示 3D 医学体数据的框架。Photon 引入了指令条件 token 调度和代理梯度传播，在训练和推理过程中自适应地减少 token，从而降低计算成本，同时减轻冗余 token 引起的注意力稀释。它结合了带有梯度恢复的自定义反向传播规则，尽管存在离散的 token 丢弃，仍能实现可微优化。为了稳定 token 压缩并确保视觉证据的可靠使用，Photon 进一步应用了正则化目标，以减轻纯文本偏见并提高可靠性。在多种医学视觉问答任务上的实验表明，Photon 在降低资源消耗并加速训练和推理的同时，实现了最先进的准确率。

## Abstract
Multimodal large language models are promising for clinical visual question answering tasks, but scaling to 3D imaging is hindered by high computational costs. Prior methods often rely on 2D slices or fixed-length token compression, disrupting volumetric continuity and obscuring subtle findings. We present Photon, a framework that represents 3D medical volumes with token sequences of variable length. Photon introduces instruction-conditioned token scheduling and surrogate gradient propagation to adaptively reduce tokens during both training and inference, which lowers computational cost while mitigating the attention dilution caused by redundant tokens. It incorporates a custom backpropagation rule with gradient restoration to enable differentiable optimization despite discrete token drop. To stabilize token compression and ensure reliable use of visual evidence, Photon further applies regularization objectives that mitigate language-only bias and improve reliability. Experiments on diverse medical visual question answering tasks show that Photon achieves state-of-the-art accuracy while reducing resource usage and accelerating both training and inference.

#### 1. 核心问题与整体含义（研究动机和背景）
在临床诊断中，3D 医学影像（如 CT、MRI）包含丰富的空间信息，但其巨大的数据量给多模态大语言模型（MLLM）带来了严峻的计算挑战。现有的方法通常采用 **2D 切片采样**（导致空间连续性丢失）或 **固定长度的 Token 压缩**（导致细微病灶被掩盖或冗余计算）。
**Photon** 的研究动机在于：如何在保留 3D 体积细节的同时，根据具体的临床指令（Instruction）自适应地减少视觉 Token，从而在不损失精度的前提下显著提升训练和推理效率。

#### 2. 方法论：核心思想与关键技术
Photon 提出了一个原生支持 3D 的 MLLM 框架，核心思想是**变长 Token 序列表示**。
*   **指令调节的 Token 调度（ITS）：**
    *   **指令感知显著性估计（ISE）：** 利用指令 Token 之间的自注意力以及指令与视觉 Token 的交叉注意力，计算视觉 Token 的重要性得分。
    *   **实例感知阈值预测器（ITP）：** 根据当前影像和指令的复杂程度，动态预测一个裁剪阈值，而非使用固定的压缩率。
*   **代理梯度传播（SGP）：**
    *   针对离散 Token 丢弃导致的不可微问题，引入了“直通估计器”（Straight-Through Estimator）。
    *   设计了基于激活值与梯度乘积的代理信号，为被丢弃或保留的 Token 提供优化方向，使 Token 选择过程在训练中可学习。
*   **正则化目标：**
    *   **软保留带（Soft Retention Band）：** 将 Token 保留率限制在稳定范围内。
    *   **鲁棒性与翻转正则化（Robustness & Flip Reg.）：** 通过扰动输入和掩码翻转，强制模型依赖视觉证据而非语言先验，减轻“幻觉”和纯文本偏见。
*   **两阶段训练：** 第一阶段仅对 3D 视觉嵌入层进行对齐；第二阶段对全模型进行任务相关的微调。

#### 7. 优点
*   **端到端可微：** 通过 SGP 解决了离散采样不可微的痛点，使 Token 裁剪策略能随主任务一起优化。
*   **动态适应：** 突破了固定压缩率的限制，实现了“按需分配”计算资源。
*   **原生 3D 支持：** 避免了切片采样带来的偏差，保留了完整的体积上下文。
*   **训练友好：** 相比于仅在推理时裁剪的方法，Photon 在训练阶段就能节省大量算力和显存。

#### 8. 不足与局限
*   **细微病灶挑战：** 在极小病灶（如 5mm 以下的结节）的精确尺寸测量上仍存在一定误差。
