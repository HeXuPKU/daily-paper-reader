## 摘要
临床决策智能体可以从重用先前的决策经验中获益。然而，许多记忆增强方法将经验存储为独立的记录，缺乏显式的关系结构，这可能导致检索噪声、重用不可靠，在某些情况下甚至比直接的大语言模型（LLM）推理性能更差。我们提出了 GSEM（基于图的自演化记忆），这是一个临床记忆框架，它将临床经验组织成双层记忆图，既捕捉了每个经验内部的决策结构，也捕捉了经验之间的关系依赖，并支持适用性感知检索以及基于在线反馈的节点质量和边权重校准。在 MedR-Bench 和 MedAgentsBench 两个基准测试中，结合两种 LLM 主干模型，GSEM 在所有基线方法中实现了最高的平均准确率，使用 DeepSeek-V3.2 和 Qwen3.5-35B 分别达到了 70.90% 和 69.24%。代码可在 https://github.com/xhan1022/gsem 获取。

## Abstract
Clinical decision-making agents can benefit from reusing prior decision experience. However, many memory-augmented methods store experiences as independent records without explicit relational structure, which may introduce noisy retrieval, unreliable reuse, and in some cases even hurt performance compared to direct LLM inference. We propose GSEM (Graph-based Self-Evolving Memory), a clinical memory framework that organizes clinical experiences into a dual-layer memory graph, capturing both the decision structure within each experience and the relational dependencies across experiences, and supporting applicability-aware retrieval and online feedback-driven calibration of node quality and edge weights. Across MedR-Bench and MedAgentsBench with two LLM backbones, GSEM achieves the highest average accuracy among all baselines, reaching 70.90\% and 69.24\% with DeepSeek-V3.2 and Qwen3.5-35B, respectively. Code is available at https://github.com/xhan1022/gsem.

#### 1. 论文的核心问题与整体含义
在临床决策中，LLM 智能体虽然强大，但在面对高风险、信息不全且具有强条件依赖的病例时，仅靠参数化知识往往不足。现有的“记忆增强”方法通常将过去的经验存储为独立的记录，存在两个核心痛点：
*   **边界失效（Boundary Failure）：** 检索时仅匹配部分条件，忽略了关键的边界约束。
*   **协作失效（Collaboration Failure）：** 检索到的多个经验之间存在冲突或缺乏共同使用的价值，导致推理逻辑混乱。
**GSEM** 旨在通过构建一个结构化的、可动态进化的双层记忆图，实现对临床经验的精准检索与可靠重用。

#### 2. 论文提出的方法论
GSEM 提出了一个包含构建、检索、进化三个阶段的框架：
*   **核心思想：** 将临床经验形式化为“适应症（Indication）”和“禁忌症（Contraindication）”，并组织成双层图结构。
*   **双层记忆图：**
    *   **实体层（Entity Layer）：** 将每个经验分解为触发条件、约束、行动、理由和结果等类型化节点，捕捉经验内部的决策流。

*   **在线反馈驱动进化：**
    *   任务完成后，根据结果反馈（标量信号）更新节点质量评分 $Q$ 和边权重 $W$。通过信用分配机制，强化对成功有贡献的经验及其关联，弱化导致失败的路径。

#### 7. 优点
*   **结构化建模：** 相比扁平的记忆库，双层图结构能更好地处理临床决策中的复杂逻辑和经验冲突。
*   **显式边界处理：** 通过“适应症/禁忌症”和适用性感知检索，有效缓解了 RAG 中的噪声重用问题。
*   **动态校准：** 无需重新训练参数，通过权重更新实现非参数化的强化学习，使系统具备持续进化的能力。
