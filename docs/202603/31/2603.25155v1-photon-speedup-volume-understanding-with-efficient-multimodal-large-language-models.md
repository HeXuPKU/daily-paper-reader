## 摘要
多模态大语言模型在临床视觉问答任务中具有广阔前景，但由于高昂的计算成本，将其扩展到 3D 成像领域仍面临挑战。先前的方法通常依赖于 2D 切片或固定长度的 token 压缩，这破坏了体积连续性并掩盖了细微的发现。我们提出了 Photon，这是一个使用变长 token 序列表示 3D 医学体数据的框架。Photon 引入了指令条件下的 token 调度和代理梯度传播，在训练和推理过程中自适应地减少 token，从而在降低计算成本的同时，减轻了由冗余 token 引起的注意力稀释。它结合了带有梯度恢复的自定义反向传播规则，以便在存在离散 token 丢弃的情况下实现可微优化。为了稳定 token 压缩并确保视觉证据的可靠使用，Photon 进一步应用了正则化目标，以减轻纯语言偏见并提高可靠性。在多种医学视觉问答任务上的实验表明，Photon 在降低资源消耗并加速训练和推理的同时，达到了最先进的准确率。

## Abstract
Multimodal large language models are promising for clinical visual question answering tasks, but scaling to 3D imaging is hindered by high computational costs. Prior methods often rely on 2D slices or fixed-length token compression, disrupting volumetric continuity and obscuring subtle findings. We present Photon, a framework that represents 3D medical volumes with token sequences of variable length. Photon introduces instruction-conditioned token scheduling and surrogate gradient propagation to adaptively reduce tokens during both training and inference, which lowers computational cost while mitigating the attention dilution caused by redundant tokens. It incorporates a custom backpropagation rule with gradient restoration to enable differentiable optimization despite discrete token drop. To stabilize token compression and ensure reliable use of visual evidence, Photon further applies regularization objectives that mitigate language-only bias and improve reliability. Experiments on diverse medical visual question answering tasks show that Photon achieves state-of-the-art accuracy while reducing resource usage and accelerating both training and inference.

#### 1. 核心问题与整体含义（研究动机和背景）
在临床医学中，3D 成像（如 CT 和 MRI）对于诊断至关重要。然而，现有的多模态大语言模型（MLLM）在处理 3D 体数据时面临严峻挑战：
*   **计算成本极高：** 完整的 3D 扫描包含海量数据，直接输入模型会导致内存溢出（OOM）。
*   **信息损失：** 传统方法要么采用 2D 切片采样（破坏空间连续性），要么使用固定长度的 token 压缩（掩盖细微病变）。
*   **指令无关性：** 现有的 token 剪枝方法通常是静态的，无法根据医生的具体提问（指令）自适应地保留关键区域。

**Photon** 的提出旨在通过**变长 token 序列**表示 3D 医学体积，在保留关键临床细节的同时，大幅降低计算开销并提升推理速度。

#### 2. 方法论：核心思想与关键技术
Photon 框架的核心在于将 3D 卷直接建模为变长 token 序列，并引入了以下关键技术：
*   **指令条件 token 调度 (ITS)：** 
    *   **指令感知显著性估计 (ISE)：** 通过计算指令 token 与视觉 token 之间的交叉注意力，识别出与当前问题最相关的视觉区域。
    *   **实例感知阈值预测器 (ITP)：** 根据图像复杂度和指令需求，动态预测每个样本的剪枝阈值，而非使用统一的保留比例。
*   **代理梯度传播 (SGP)：** 
    *   针对离散 token 丢弃不可微的问题，Photon 设计了自定义反向传播规则。
    *   利用一阶泰勒展开（激活值与梯度的乘积）作为代理信号，使模型能够学习如何根据任务目标优化 token 的保留概率。
*   **正则化目标：** 
    *   **软保留带 (Soft Retention Band)：** 将 token 保留率限制在稳定范围内。
    *   **鲁棒性与翻转正则化 (Robustness & Flip Reg.)：** 通过扰动实验和掩码翻转，强制模型依赖视觉证据而非文本先验，减轻“幻觉”和纯语言偏见。
*   **两阶段训练：** 第一阶段仅对 3D 视觉嵌入层进行轻量级对齐；第二阶段对全模块进行任务相关的微调。
